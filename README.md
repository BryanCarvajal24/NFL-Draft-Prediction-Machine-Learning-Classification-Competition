# NFL Draft Prediction - Machine Learning Classification Competition

## Overview
This repository contains the solution for the NFL Draft Prediction Competition (GCI World 2026).

The project includes:

- Exploratory Data Analysis (EDA)
- Data Preprocessing
- Feature Engineering
- Machine Learning Classification Models
- Model Evaluation using AUC-ROC
- Competition Submission Generation

## Objective

Predict whether a player will be selected in the NFL Draft using physical measurements, athletic performance metrics, and position-related information.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Random Forest
- CatBoost
- LightGBM
- Matplotlib
- Jupyter Notebook

## Evaluation Metric

- AUC-ROC
